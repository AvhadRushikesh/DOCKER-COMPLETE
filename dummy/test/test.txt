This command allow us to copy something / anything in container without restarting the container or rebuilding the image


docker cp dummy/. containerName:/test --------- cp = allows us to copy file into running container
                                                copy all the files from dummy folder (dummy = folderName, /. = all, 
                                                test = this path will be created if it does not exist in container
                                                /test = inside of your container you want to copy to)


docker cp containerName:/test dummy ----------- If we delete the test file from our project that we copy in last command from dummy folder that we copied in container's test path by
                                                running this command we will get it back

docker cp containerName:/test/test.txt dummy -- copy the test file from container as test.txt file in test folder under dummy folder
